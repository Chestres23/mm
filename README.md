# Dia de la Mujer - Web Components

Pagina web moderna y responsive hecha con Web Components y localStorage.

## Secciones

- Inicio: portada con mensaje "Feliz Dia de la Mujer"
- Mensajes: frases romanticas dedicadas
- Sorpresa: contador de cuantas veces duerme y cuantas veces se enoja

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Tecnologias

- Vite
- Web Components (Custom Elements)
- localStorage
